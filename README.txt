album file allowed structure:
- 1 png/jpg/jpeg file containing album's image (optional)
- song files (mp3/wav/flac) (optional)
- the album file's name must follow this format: "{artist} - {album name}" (must)

for control:
main menu page:
- arrow up and down: navigating
- round button (choose button): choose

album page:
- arrow left and right: navigating
- round button: choose

track choosing:
- arrow up and down: navigating
- arrow right: edit track
- arrow left (only when accessed through an album): back to choosing album
- choose button: play track

track editing:
- arrow left: back to track choosing
- choose: apply the option

the letter (main menu): to back to the main menu
